IEEE-Format Paper / September 12, 2025
PrivExpensIA: Observational Study of a Multi-Agent Orchestration Framework
Based on Local Claude CLI with Email-mediated Communication
Abstract

This paper documents the experimental deployment of a local multi-agent orchestration system, nicknamed La Poste de Moulinsart, applied to the development of a production-ready iOS application (PrivExpensIA v1.0). Unlike cloud-based agent frameworks (e.g., CrewAI, LangGraph, AutoGen), this architecture leverages parallel Claude CLI instances orchestrated via email-based communication and tmux integration.

Results indicate that the system autonomously executed multiple development sprints, delivered tangible artefacts (code, i18n files, heuristics, training material), and achieved production-level performance (>95% accuracy, <400ms latency, 0 crashes/1000 ops). The observed behaviour suggests a robust orchestration pipeline capable of sustaining long-term context and iterative refinement beyond the limits of stateless API-driven approaches.

Communication

Local SMTP server routes structured emails between agents. Asynchronous, auditable, persistent — unlike ephemeral API calls.

Context

200k tokens per agent. No resets between tasks. Each CLI instance maintains full project context throughout the session.

Observability

WebSocket + Vue.js dashboard with real-time monitoring. SQLite event database tracks every action, email, and state change.

Process

PRD injected at project root. Nestor decomposes into sprints, dispatches tasks, validates outcomes, triggers next sprint — autonomously.

What four agents built in one session

PrivExpensIA — A Complete Expense Tracker

A production-ready iOS application that competes with Expensify. Built autonomously by 4 AI agents during a single orchestration session (2h20 runtime). Features include:

  • AI-powered OCR receipt scanning (Vision Framework + Qwen2.5-0.5B on-device)
  • Smart expense categorization with Swiss VAT handling
  • Multi-currency support
  • 8 languages (FR, DE, IT, EN, ES, JA, KO, SK) including Swiss variants
  • Statistics dashboard with spending insights
  • 100% on-device processing — zero cloud dependencies
  • Zero crashes across 1,000 operations

Conclusion
La Poste de Moulinsart demonstrates a viable, robust alternative to cloud agentic frameworks. By combining CLI agents, email orchestration, and persistent observability, the system achieved production-level artefacts in less than 48h runtime. This suggests that local-first, CLI-based multi-agent systems can outperform stateless API-based approaches for complex, iterative software engineering tasks.
