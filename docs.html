About Moulinsart
La Poste de Moulinsart was the first multi-agent orchestration system built by Infinity Cloud in September 2025 — two months before OpenClaw's first commit. Four AI agents coordinated via tmux sessions, communicated through an email-based postal system, and built production software autonomously. The system is documented in an IEEE-format research paper dated September 12, 2025.
